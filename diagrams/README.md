# Architecture Diagrams

This directory contains Mermaid diagram files for the capstone project architecture.

## Files

- **system-architecture.mmd** - High-level system architecture showing components and their interactions
- **data-flow.mmd** - Detailed data flow sequence diagram showing the complete data lifecycle

## Viewing the Diagrams

### Option 1: GitHub (Recommended)
GitHub automatically renders Mermaid diagrams. Simply view these files on GitHub to see the visualizations.

### Option 2: VS Code
Install the "Markdown Preview Mermaid Support" extension to preview diagrams locally.

### Option 3: Online Mermaid Editor
1. Visit [Mermaid Live Editor](https://mermaid.live/)
2. Copy the contents of any `.mmd` file
3. Paste into the editor to view and export
